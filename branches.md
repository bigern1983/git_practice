## Git Branches

list all the branches in a repo
```
git branch
```

Create a new branch

```
git branch <new branch name>
```

Delete a branch - this is a safe delete, does not let you delete branch if it has unmerged changes. 

```
git branch -d <branch>
```

Delete a branch - even if it has unmerged changes. 

```
git branch -D <branch>
```

Rename current branch
```
git branch -m <branch name>
```

List all remote branches
``` 
git branch -a 
```
