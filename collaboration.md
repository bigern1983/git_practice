# Git Collaboration 

## Process 

1. Fetch and merge changes from the remote 
   ```
   git fetch
   git merge origin/master
   ```
2. Create a branch to develop on `git checkout -b develop`
3. Do some coding and commit changes `git commit -m "changes"`
4. Fetch and merge with remote again 
5. Push new stuff to remote

## Commands

* `git clone` Creates a local copy of a remote. 
* `git remote -v` Lists a git projects remotes. 
* `git fetch` Fetches work from remote into the local repo.
* `git merge origin/master` Merges origin/master into your local branch. 
* `git push origin <branch_name>` pushes a local branch onto the origin remote.
